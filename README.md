# 🐱‍👤 Buscador de Pokémon

## Descripción

Este proyecto consiste en una aplicación web desarrollada con **HTML, CSS y JavaScript** que permite buscar información de cualquier Pokémon utilizando la **PokeAPI**.

La aplicación consume datos desde una API pública y muestra la información de forma dinámica.


## Funcionalidades

- Buscar un Pokémon por su nombre.
- Buscar presionando el botón **Buscar**.
- Buscar presionando la tecla **Enter**.
- Mostrar:
  - Imagen oficial
  - Nombre
  - ID
  - Tipo
  - Altura
  - Peso
  - Habilidades
  - Estadísticas principales
- Mostrar un mensaje de error cuando el Pokémon no exista.

## Cómo ejecutar el proyecto

1. Descargar o clonar el proyecto.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo **index.html** en cualquier navegador moderno.
4. Escribir el nombre de un Pokémon (por ejemplo: pikachu, charizard, bulbasaur).
5. Presionar **Buscar** o la tecla **Enter**.
